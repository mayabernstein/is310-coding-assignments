## Command Line Data Curation

### Directions
Through this application, you'll be prompted to input some the Olympic records for swimming. After including information like the name of
the Olympian, country they represented, the swimming event, the recorded time, and Olympic the records were made, you will give the option to check if the inputs are accurate. If you would like to fix the information you inputted, you would be able to do so. After finishing inputting the records, you will be able to see the appended information in the table. 

The Internet will be your best friend if you are unsure the exact information to include. 

I hope this application would bring you to new information you don't know about swimming like when these records made, from what countries, etc. 

The point of this activity is to have fun, so even if the information is not accurate, no worries!
